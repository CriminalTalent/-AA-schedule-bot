# 스케줄 봇

마스토돈 기반 텍스트 RPG 스케줄 봇입니다.
플레이어는 턴마다 행동 3개를 제출하고, GM은 전체 현황 관리와 수치 조정을 할 수 있습니다.

---

## 시작하기

### 요구 사항

- Node.js 18 이상
- 마스토돈 계정 및 액세스 토큰
- Google Cloud 서비스 계정
- Google Sheets 스프레드시트 (공유 DB)

### 설치
```bash
npm install
```

### 환경 변수
```env
MASTODON_URL=https://your.instance.url
BOT_TOKEN=your_mastodon_access_token
GM_ACCOUNT_ID=gm_account_id
MAX_TURNS=24

GOOGLE_SHEET_ID=your_google_sheet_id
GOOGLE_SERVICE_ACCOUNT_JSON={"type":"service_account","project_id":"..."}
```

GM_ACCOUNT_ID는 마스토돈 계정 숫자 ID입니다. 계정 설정 또는 API에서 확인할 수 있습니다.

### 실행
```bash
node bot.js
```

PM2 사용 시:
```bash
pm2 start bot.js --name schedule-bot
pm2 save
```

모든 명령은 스케줄 봇 계정을 멘션한 뒤 대괄호 `[ ]` 안에 입력합니다.
@스케줄봇 [명령어]

### 상태 확인

| 명령 | 설명 |
|---|---|
| `[상태]` | 현재 수치 / 나이 / 턴 / 컨디션 / 소지금 출력 |

출력 예시:
[홍길동] 초기 성장기 / 10세 / 3턴

### 스케줄 제출
@스케줄봇 [스케줄/행동1] [스케줄/행동2] [스케줄/행동3]

한 번에 행동 3개를 제출합니다.

제출 규칙:
- 행동은 정확히 3개여야 합니다.
- `무사수행`은 턴당 1회만 선택할 수 있습니다.
- 같은 행동을 중복 선택하면 효과에 50% 페널티가 적용됩니다.
- 이미 제출한 턴에는 재제출할 수 없습니다.
- MAX_TURNS 초과 시 제출이 불가합니다.

제출 가능한 행동은 Google Sheets Actions 탭에서 관리합니다.

행동 카테고리:

| 카테고리 | 설명 |
|---|---|
| 교육 | 골드 지출, 수치 상승 |
| 아르바이트 | 골드 수입, 일부 수치 변화 |
| 휴식 | 스트레스 감소, 체력 회복 |
| 사교 | 매력·사회성·평판 상승 |
| 무사수행 | 무사수행 봇 연계, 별도 진행 |

무사수행 포함 시 제출 후 DM으로 안내가 발송됩니다.

결과는 공개 게시됩니다.

## GM 명령

GM 계정(`GM_ACCOUNT_ID`)만 사용할 수 있습니다.

### 전체 현황

| 명령 | 설명 |
|---|---|
| `[현황]` | 전체 플레이어 제출 현황 출력 |

출력 항목: 완료/대기 여부 / 턴 / 무사수행 대기 여부 / 스트레스 / 위험도

---

### 플레이어 상세

| 명령 | 설명 |
|---|---|
| `[상세]` | 전체 플레이어 상세 출력 |
| `[상세/이름]` | 특정 플레이어 상세 출력 |

출력 항목: 공개 수치 / 숨김 수치 / 골드 / 인벤토리 / 장착 현황

---

### 강제 진행

| 명령 | 설명 |
|---|---|
| `[강제진행]` | 미제출 플레이어를 다음 턴으로 강제 이동 |

해당 턴 행동 없이 턴 번호만 증가합니다.

---

### 골드 지급
[골드지급/플레이어명/금액]

양수면 지급, 음수면 차감입니다. 마이너스 골드가 허용됩니다.

---

### 수치 조정
[수치조정/플레이어명/수치명/값]

조정 가능한 수치:

| 공개 수치 | 숨김 수치 |
|---|---|
| 지능 / 매력 / 체력 / 감성 / 사회성 | 도덕성 / 야망 / 위험도 / 의존성 / 스트레스 / 평판 / 전투 |

값은 0~100으로 자동 클램프됩니다.

---

[공지/내용]

공개 타임라인에 공지를 게시합니다.

## Google Sheets 연동

### Actions 탭

행동 데이터를 관리합니다. 봇 기동 후 5분 캐시됩니다.
행동명 | 카테고리 | 최소나이 | 골드 | 효과 | 설명

| 컬럼 | 설명 |
|---|---|
| 행동명 | 명령어에서 사용하는 이름 |
| 카테고리 | 교육 / 아르바이트 / 휴식 / 사교 / 무사수행 |
| 최소나이 | 이 나이 미만이면 선택 불가 |
| 골드 | 양수=수입, 음수=지출 |
| 효과 | JSON 문자열. 예: `{"지능":3,"스트레스":-2}` |

### Players 탭

플레이어 데이터가 자동으로 저장됩니다.
accountId | name | 지능 | 매력 | 체력 | 감성 | 사회성 | 도덕성 | 야망 | 위험도 | 의존성 | 스트레스 | 평판 | 전투 | 골드 | 턴 | 인벤토리 | 장착
